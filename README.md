## How to Use

1. Open `index.html` in a web browser
2. Use the buttons to interact:
   - **Scramble** - Mix up the cube randomly
   - **Solve** - Auto-solve the cube by reversing the scramble moves
   - **Reset** - Reset to the solved state

## Controls

- **Drag** - Rotate the cube view
- **Scroll** - Zoom in/out
- **Buttons** - Scramble, Solve, Reset

## Technical Details

- Built with [Three.js](https://threejs.org/) (loaded via CDN)
- Single HTML file - no build process required
- Standard Rubik's cube colors:
  - White (Up)
  - Yellow (Down)
  - Red (Right)
  - Orange (Left)
  - Blue (Front)
  - Green (Back)
